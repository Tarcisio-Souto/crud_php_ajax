A proposta principal deste projeto é entregar um sistema no formato SPA (single page application)
trabalhando com algumas técnicas, principalmente da linguagem PHP, do Ajax e do JQuery.
Como exemplo, há um CRUD envolvendo o cadastro de funcionários e anotações, e também dos dados
da empresa.

#HTML
#CSS
#Bootstrap
#Bootbox
#Javascript
#Ajax
#JQuery
#PHP

OBS.: recomendo a utilização do XAMPP, principalmente porque há a base de dados
para ser utilizada. Aliás, o script para a criação do banco (empresa.sql) está na
pasta do projeto.
